Pre condition:

1. Clone the application from below github link
    https://github.com/buildit/acceptance-testing to local 
2. Run the Weather Forecast application by using below command (User should be in acceptance-testing-master folder)

npm install

npm run develop

Set the local host path of application running in weatherForecast.js file and run the script.

Steps to run the script:

Install node js
Install gulp using below command npm install -g gulp
Install Selenium-cucumber-js by running command npm install selenium-cucumber-js --save-dev

Run script: gulp execute --file=weatherForecast

Note:
Node version used - v10.14.2
